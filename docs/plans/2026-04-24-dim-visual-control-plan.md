## 1. 新增字段

| 字段 | 类型 | `$` 变量 | DXF code | Default | 语义 |
|---|---|---|---|---|---|
| `dim_just` | `i16` | `$DIMJUST` | 70 | `0` | 标注文字水平对齐（0 居中 / 1 第一延伸线 / 2 第二 / 3 在第一上方 / 4 在第二上方） |
| `dim_sd1` | `i16` | `$DIMSD1` | 70 | `0` | 抑制第一标注线（0 显示 / 1 隐藏） |
| `dim_sd2` | `i16` | `$DIMSD2` | 70 | `0` | 抑制第二标注线 |
| `dim_se1` | `i16` | `$DIMSE1` | 70 | `0` | 抑制第一延伸线 |
| `dim_se2` | `i16` | `$DIMSE2` | 70 | `0` | 抑制第二延伸线 |
| `dim_soxd` | `i16` | `$DIMSOXD` | 70 | `0` | 抑制外部延伸线标注 |
| `dim_atfit` | `i16` | `$DIMATFIT` | 70 | `3` | 文字与箭头空间不足时的放置方式（0 文字和箭头 / 1 箭头 / 2 文字 / 3 任意可行） |
| `dim_azin` | `i16` | `$DIMAZIN` | 70 | `0` | 角度标注零压缩 bitfield |
| `dimaunit` | `i16` | `$DIMAUNIT` | 70 | `0` | 角度单位格式（已在 DIM tier-1 从 header 覆盖，这里补完） |
| `dim_tix` | `i16` | `$DIMTIX` | 70 | `0` | 强制文字在延伸线内（0 否 / 1 是） |
